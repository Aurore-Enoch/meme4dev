# meme4dev

## backend
1. Se placer dans le dossier backend
```bash
cd backend
```

2. Installer les nodes_modules 
```bash
npm install
```

3. Lancer le projet
```bash
npm start
```

## frontend
1. Se placer dans le dossier frontend
```bash
cd frontend
```

2. Installer les nodes_modules
```bash
npm install
```

3. Lancer le projet
```bash
npm run dev
```
